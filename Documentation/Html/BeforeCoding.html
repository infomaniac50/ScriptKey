Before you start coding there are some things you need to know. In order to find your methods the namespace and the class called UserCode must not be changed

When using triggers like the one seen here

You must make sure your trigger methods are declared public

Because trigger methods are the end of the line they should all return void

and since there is nothing to pass to them when the trigger fires they should have zero parameters
