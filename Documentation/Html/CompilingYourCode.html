Compiling Your Code

Operation ››

In order to run your code you must first compile it by clicking the compile button.



Then you should see that it compiled successfully



If by some chance your code didn't compile like this here.



The compiler status will change and a detail of the error will be shown. You will also be unable to run your code until the errors are fixed.



There are other compiler status changes that do not affect your code but merely serve as a reminder to clean up.
